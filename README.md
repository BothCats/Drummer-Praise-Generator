### 🥁 Drummer Praise Generator

> **献给那位在舞台上闪闪发光的鼓手。**

这是一个基于 Web Audio API 和 HTML5 Canvas 开发的单页面应用。它不仅仅是一个夸夸生成器，更是一次**理性代码与感性节奏**的碰撞实验。

#### ✨ 功能亮点

-   **🎵 实时音频合成**：不依赖任何外部 MP3 文件，使用 JavaScript 代码实时合成底鼓、军鼓和踩镲的声音。
-   **🎶 动态节奏编排**：内置多种鼓手常用节奏型（如摇滚、三连音、切分音），每次点击都会随机演奏一段精彩的鼓点。
-   **✨ 梦幻星空特效**：基于 Canvas 的粒子系统，鼠标移动时会产生交互式的光点排斥与连线效果，模拟舞台聚光灯下的氛围。
-   **💖 INFJ 专属语录**：精选适合细腻、敏感、有才华的 INFJ 人格的赞美文案。

#### 🛠️ 技术栈

-   **HTML5 / CSS3**: 玻璃拟态设计，渐变背景动画。
-   **Vanilla JavaScript**: 原生 ES6+ 语法，零依赖。
-   **Web Audio API**: 核心音频引擎，负责生成 Kick, Snare, Hi-Hat 音效。
-   **Canvas API**: 负责高性能的粒子动画渲染。

#### 🚀 如何运行

本项目为纯静态页面，无需构建工具。

1. 克隆本仓库：

   ```bash
   git clone https://gitee.com/cool_LIH/drummer-praising.git
   ```

2. 直接在浏览器中打开 `index.html` 即可体验。
